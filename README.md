# رُواة

## الوصف
تطبيق ثقافي سعودي تفاعلي لاستكشاف التراث والثقافة المحلية بالذكاء الاصطناعي. يوفّر خريطة تفاعلية للمواقع الثقافية، ومساعدًا ذكيًا اسمه "الراوي" يحكي عن التراث واللهجة والعادات، مع نظام نقاط لتحفيز الاستكشاف ودعم كامل للعربية وإمكانية الوصول.

## المزايا
- خريطة تفاعلية للمواقع الثقافية والتراثية
- "الراوي" — مساعد ذكي يحكي عن التراث واللهجة والعادات
- نظام نقاط لتحفيز الاستكشاف
- دعم كامل للعربية (RTL) وإمكانية الوصول
- رفع المتاجر الثقافية لتظهر على الخريطة

## التشغيل
```bash
npm install
npm run dev
```

## التصميم

## بيئة التطوير
- **Visual Studio Code** — محرر الكود

- **React** — مكتبة بناء الواجهات
- **Vite** — أداة البناء السريعة
- **Tailwind CSS** — إطار التنسيق
- **React Router** — التنقل بين الصفحات
- **Google Maps API** — الخرائط التفاعلية
